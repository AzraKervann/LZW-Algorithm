# LZW Algorithm

This project implements the LZW (Lempel-Ziv-Welch) compression algorithm in Java.

## Features
- Encoding (Compression)
- Decoding (Decompression)
- Lossless data compression

## How It Works
LZW builds a dictionary of input sequences and replaces repeated patterns with shorter codes.

## Usage
Run the main class to test compression and decompression.
